# fmcg-sales-data-pipeline

Project goal: Build a production-style data engineering pipeline from an Excel dataset containing FMCG sales data.

Project phases (in order):

1. Project setup
2. Environment setup
3. Explore the Excel workbook
4. Design the star schema
5. Build the Python ETL (validation, cleaning, incremental loading, logging)
6. Build dbt staging and mart models
7. Write SQL business queries
8. Orchestrate everything with Airflow
9. Containerize with Docker
10. Documentation and presentation
